#MOVERSE
Para transformar
KITESO es un proyecto desarrollado para Web en la clase de Diseño de Software con el fin de dar un mejor servicio a los estudiantes del ITESO, proporcionandoles la facilidad de organizar su horario y conocer sus maestros, así también, esta aplicación web brinda el servicio de poder transladarte a tus salones de clase por el mejor y mas corto camino, para los estudiantes de nuevo ingreso y así se pueda dar uso de todas las diferentes areas e instalaciones que el ITESO como Universidad nos brinda.
